Instructions: HTML Practice & CSS Intro

HTML (Content)

For this task, you will be editing the index.html file. Please navigate to the HTML Reference on the CS 130 course website. Use these resources to help you create a media gallery that has some kind of theme, for example: "dogs," "nature," "food", "art", "life during COVID-19," etc. If you're not feeling creative, then create a media gallery to showcase things you love about the place you're currently living.

Please include the following components in your gallery:

  1. A title
  2. A heading (and some sub-headings)
  3. A paragraph describing the media you're featuring
  4. A few images
  5. An IFrame (embedded Tweets, songs, videos, etc.)
  6. Some hyperlinks to related content on the internet

CSS (Style)

For this task, you will be editing the style.css file. Please navigate to the CSS Reference on the CS 130 course website. Add some style to your media gallery!
